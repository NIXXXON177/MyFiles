# Отчёт по тестированию функции

## Область определения

**Функция:**

y = √(x² - 16) + (x + 24)/(x - 9)

**Условия:**
- Подкоренное выражение должно быть неотрицательным: x² - 16 ≥ 0
- Знаменатель дроби не должен обращаться в ноль: x - 9 ≠ 0

**Итоговая область определения:**
- x ≤ -4
- или 4 ≤ x < 9
- или x > 9

## Техники тест-дизайна и тест-кейсы

### Анализ классов эквивалентности

**Классы эквивалентности:**

1. Корректное отрицательное значение: x < -4 (например, x = -10, -5)
2. Левая граница: x = -4
3. Некорректная область: -4 < x < 4
4. Правая граница: x = 4
5. Корректное положительное значение: 4 < x < 9 (например, x = 5, 8)
6. Особое значение (деление на ноль): x = 9
7. Корректное значение больше 9: x > 9 (например, x = 11)
8. Некорректный ввод: не число ("abc", "!")
9. Пустое значение

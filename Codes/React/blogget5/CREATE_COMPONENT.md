# Создание компонентов через Hygen

## Использование

Запустите команду:

```bash
npm run component
```

Или напрямую:

```bash
npx hygen create component
```

## Шаги при создании компонента:

1. **Где создать компонент?** - Выберите:

   - `components` - для компонентов приложения
   - `UI` - для переиспользуемых UI компонентов

2. **Имя компонента** - Введите имя (например: `Svg`)

3. **Вложенная директория?** - Опционально, если нужна вложенность (например: `Header/Auth`)

4. **Добавить файл стилей?** - Да/Нет (по умолчанию: Да)

## Примеры:

### Создать компонент Svg в UI:

```
Где создать компонент? → UI
Имя компонента → Svg
Вложенная директория? → (оставить пустым)
Добавить файл стилей? → Нет (для компонента Svg стили могут не понадобиться)
```

### Создать компонент Button в UI:

```
Где создать компонент? → UI
Имя компонента → Button
Вложенная директория? → (оставить пустым)
Добавить файл стилей? → Да
```

### Создать компонент в подпапке:

```
Где создать компонент? → components
Имя компонента → Auth
Вложенная директория? → Header
Добавить файл стилей? → Да
```

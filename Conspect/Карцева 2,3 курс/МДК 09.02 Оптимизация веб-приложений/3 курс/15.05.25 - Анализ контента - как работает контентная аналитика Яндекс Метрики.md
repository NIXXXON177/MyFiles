# Для чего нужна яндекс метрика?

Яндекс.Метрика выдает подробный статистический анализ контента на сайте: какие статьи генерируют трафик, из каких каналов приходят посетители на страницы, какие посты популярны у аудитории, дочитывают ли люди материалы. Статистика по контенту помогает понять, интересуют ли темы ЦА, какие рубрики приносят конверсию, а что стоит исключить.

## Получение статистики по контенту

Главное условие в настройках счетчика Метрики необходимо подключить опцию «Контентная аналитика». Для этого выбираем соответствующий счётчик и кликаем по кнопке «Настройка»:

Если на сайте одновременно два типа микроразметки, то укажите предпочтительный вариант. Для турбо-страниц выбирать разметку не нужно.

Итак, главные условия для сбора статистики по контенту:
	- должен быть установлен один счетчик, как для сайта, так и турбо-страниц;
	- настроены одинаковые идентификаторы и заголовки для стандартных страниц ресурса и ускоренных;
	- чтобы идентифицировать рубрику, необходимо использовать разметку breadcrumblist.

## Проведение анализа контента с помощью метрики

Посмотреть контентную аналитику можно в разделе «Контент». Пользователям доступны два варианта отчетов: сводные и стандартные.

1. ==**«Сводка»**==

Во вкладке «Сводка» находится краткая статистика по контенту за сутки.

Данные помогают оценить поведение пользователей на сайте конкретно в эту минуту:
	- какая активность у аудитории;
	- какие публикации пользуются успехом у посетителей;
	- откуда пришли пользователи на страницу (из социальных сетей, органического поиска, РРС и так далее).

В «Сводке» отображаются следующие показатели:
	- количество посетителей сайта;
	- какое число людей посмотрело материал и сколько публикаций посмотрел один пользователь, если на странице пара-тройка статей;
	- время просмотра контента;
	- рециркуляция — процент пользователей, которые после прочтения одной публикации перешли на другую страницу;
	- распределение просмотров в зависимости от канала перехода.

2. ==**«Материалы»**==

В этом отчете можно получить статистику по опубликованному контенту и сравнивать показатели публикаций между собой. Для просмотра данных нужно перейти из раздела «Контент» во вкладки «Сводные отчеты» — «Материалы».

![[Pasted image 20250515103219.png]]

Отчеты «Материалы» позволяет узнать:
	1. Насколько контент полезен для ЦА
	2. Узнать, с каких устройств пользователи чаще читают материалы
	3. Лучше узнать целевую аудиторию
	4. Посмотреть, на какие публикации пользователи переходят после прочтения конкретной статьи

3. ==«Рубрики»==

В отчете можно посмотреть статистику по рубрикам корпоративного блога.

Чтобы увидеть показатели по контенту нужно открыть вкладку «Сводные отчеты» и кликнуть по разделу «Рубрики».

![[Pasted image 20250515103520.png]]

В отчете данные представлены следующим образом:
	- По сайту. Это обобщенная статистика контента по рубрикам. Аналитик сможет определить популярные разделы блога, посмотреть, из каких каналов приходят посетители (внутренние переходы, из социальных сетей, поисковиков и так далее), определить процент аудитории, которая перешла из конкретной рубрики в другую или дальше на сайт.
	- Для конкретной рубрики. Анализ контента помогает определить топовые материалы. С учетом интересов аудитории можно составить или откорректировать контент-план.

4. ==«Авторы»==

Аналитик может посмотреть статистику по конкретному автору, нажав на имя в перечне копирайтеров или журналистов. По каждому сотруднику высветится сумма публикаций и количество прочтений материалов.

Используйте фильтр по длине контента в Метрике, чтобы найти, кто из авторов пишет короткие новости, а кто - лонгриды.

Чтобы данные отображались корректно, необходимо, чтобы публикации имели правильную микроразметку — Schema.org или Open Graph.

5. ==«Тематики»==

Анализ контента по «Тематикам» поможет понять интересы аудитории. Важно, чтобы на страницах присутствовала микроразметка, иначе данные будут отображаться некорректно. Здесь же можно посмотреть популярных авторов в конкретной тематике, уровень вовлеченности посетителей, а также данные о тех, кто просматривал материал - пол и возраст посетителей.

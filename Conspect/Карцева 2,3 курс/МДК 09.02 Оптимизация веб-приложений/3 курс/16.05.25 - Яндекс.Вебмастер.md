# Для чего нужен Вебмастер

Яндекс.Вебмастер — инструмент, с помощью которого можно контролировать работу сайта и его представление в поиске.

**Основные функции сервиса**
	- диагностика ошибок сайта, которые влияют на его доступность для пользователей и на позиции в поисковой выдаче;
	- представление подробной статистики по индексации страниц поисковыми роботами;
	- сбор и возможность редактирования важной информации о сайте — региональность, зеркала адресов и пр.;
	- контроль размещения внешних ссылок;
	- инструменты для проверки основных технических параметров сайта;
	- диагностика работы турбо-страниц для контентных проектов и интернет-магазинов.

# Последовательность действий для добавления сайта в Яндекс.Вебмастер

1. Добавление адреса сайта
2. Подтверждение прав на указанный сайт
	a) Добавление сгенерированного мета-тега на главную страницу сайта (в HTML-код)
	b) создание файла в корневой папке сайта с указанным именем
	c) Добавление записи в DNS домена

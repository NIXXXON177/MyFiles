# 14.02.25 - продолжение

**Дорожная карта формируется следующим образом:**
	1. Выделяются группы факторов, которые влияют на продвижение сайта. Обычно это контентные факторы, ссылочные факторы и факторы внутренней оптимизации сайта. Иногда выделяют поведенческие факторы и факторы-соцсигналы. Кроме того, факторы группируются по некой общности SEO-средств, которые призваны улучшать значения этих факторов.
	2. "Слияние" групп факторов и SEO-средств образуют несколько различных фронтов SEO-работ. Внутри каждого фронта SEO-работ решается **многокритериальная задача**, на вход которой **поступают тип сайта** (включающий тип сайта по базовой стратегии продвижения и возраст сайта), **тип целевой аудитории** и иногда **фазу взаимодействия с целевой аудиторией**.

**Критериями в данной многокритериальной задаче являются:**
	- основное действие (устранение опасности / полезность надежная / полезность конвенционально-кумулятивная);
	- быстрота получения результата (быстрый результат / долгий по действию / долгий по результату / долгий и по действию, и по результату);
	- величина получаемого результата (прорывной результат (разы и порядки) / средний результат (десятки процентов) / малый результат (доли процента и проценты)).

**Такую дорожную карту можно использовать в качестве примера, как отправную точку, как нулевое приближение:**
	1. Чистка ссылочного профиля - устранение возможности отрицательного результата (в противоположность нацеленности на положительный результат дальнейших пунктов);
	2. Техническое SEO - быстрые и надежные результаты дублирующиеся / отсутствующие тайтлы, метаописания, заголовки, редиректы на релевантные страницы по результатам анализа семантического ядра;
	3. Контентное SEO - надежные, но не быстрые результаты. Работа над контентом высокочастотников (довольно надежно, но не быстро). Поправить очевидные семантические нестыковки в тегах и метатегах (надежно, но долго, т.к. много ручной работы);
		- Ссылки с сайтов-партнеров - конвенционально-кумулятивный результат;
		- Техническое SEO - конвенционально-кумулятивные результаты. Неиндексация пассивных параметров урла; редиректы со "страниц-призраков" и т.д.;
		- Контентное SEO -  конвенционально-кумулятивные результаты. Публикация большого количества ценных материалов на своем сайте и сторонних сайтах (раньше, потому что долго запускать; чтобы время работало на нас). Уточнение тайтлов и других тегов. 

# Алгоритмы продвижения сайтов "в будущем"

Чтобы сохранить полученный результат в поисковой выдаче, нужно внимательно следить за трендами развития поисковых систем. Для этого имеет смысл делать следующее:
	1. Обращать внимание на сервисы, помогающие автоматически генерировать статьи из крупных текстовых кусков ("больших" суждений).
	2. Обращать внимание на сервисы, помогающие автоматически генерировать статьи из отдельных суждений, создавая детализированное покрытие предметной области. Детализировать представление предметной области стоит до того, как статьи начнут "мешать" друг другу продвигаться в поиске.
	3. Обращать внимание на сервисы, помогающие автоматически генерировать из суждений уже не веб-страницы, а качественно новые объекты типа Яндекс.Островов.

Способы подключения скрипта к странице:
	- Инлайн - \<script>alert('Hello world')\</script>
	- Из сети (используется для библиотек) - \<script src='ссылка на что-то в инете'>
	- Из локального файла - \<script src='название JS файла'>\</script>

Вывод в консоль:
	- console.log() - вывод
	- console.info() - вывод
	- console.warn() - предупреждение
	- console.error() - ошибка

Создание переменных:
	- var - можно объявить несколько раз, нет области видимости, НЕ ИСПОЛЬЗОВАТЬ
	- let - нельзя объявить несколько раз
	- const - константа

Комментарии - строчные (// …) или многострочные (/*…*/)

Виды названий переменных:
	- lowerCamelCase - локальные переменные, функции
	- UpperCamelCase - прототипы и объекты
	- UPPER_SNAKE_CASE - глобальные переменные

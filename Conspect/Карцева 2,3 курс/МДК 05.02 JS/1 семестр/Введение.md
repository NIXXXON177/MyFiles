==JavaScript== - объектно-ориентированный язык для веб-разработки.

Чаще всего используют фреймворки (библиотеки для расширения возможностей языка, ускоряющие создание ПО)

Задачи:
	1 - Внести элементы автоматизации в браузере и обеспечить обработку событий
	2 - Создание интерактивных элементов
	3 - Создание аккордеонов, слайдеров, и т.д.

JavaScript имеет типы и операторы, встроенные объекты и методы. Синтаксис происходит от языков Java и C. Функциональность классов обеспечивается прототипами объектов, а не классами.

Клиенткой частью приложения называют ту часть, с которой напрямую взаимодействует конечный пользователь.
